# Tetris

This is a C++ implementation of the game Tetris using the [Raylib](https://github.com/raysan5/raylib) library.

## How to run the code (Linux)

Clone this repo
```bash
git clone https://github.com/Soyadrul/tetris
```

Configurate and generate the Makefile
```bash
mkdir build && \
cmake ..
```

Compile
```bash
make
```

Run the project
```bash
./tetris
```
